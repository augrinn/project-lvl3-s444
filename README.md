# Загрузчик страниц

## Установка

    npm install page-loader-au

## Использование

    Usage: page-loader-au [options]

    Downloads the page and saves it locally.

    Options:
        -V, --version       output the version number
        -o, --output [dir]  Output dir
        -h, --help          output usage information
